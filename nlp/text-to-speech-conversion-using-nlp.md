# Text To Speech Conversion Using NLP

Text To Speech Conversion Using NLP means converting text to the voice speech using NLP.&#x20;



pyttsx4: A Text-To-Speech Converter for Python

A Python-based alternative for text-to-speech conversion that works offline

Install pyttsx4 using pip ([https://pypi.org/project/pyttsx4/](https://pypi.org/project/pyttsx4/)) with the following command.

```sh
// Using pip
pip install pyttsx4
```

For **Conda** users also, the above command works.
